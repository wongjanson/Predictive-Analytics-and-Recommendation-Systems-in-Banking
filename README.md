## 🔧 How to Use the Application

The app has three main parts:

### 1. Loan Default Prediction

You can upload your bank’s loan data using common file types like .csv. The app will show the chance of default on each loan. It uses a decision tree and gradient boosting algorithms.

### 2. Customer Segmentation

This tool analyzes your customers to identify groups with similar features. It uses K-means and DBSCAN clustering methods. You will see grouped customer lists with insights on behavior patterns.

### 3. Product Recommendation

This feature suggests bank products that fit each customer’s profile. It uses the Surprise library and KNN classifier. You can select a customer and see recommended loans, savings accounts, or credit cards.
